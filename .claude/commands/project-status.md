# Status projektu

Pokaż aktualny stan projektu Godot.

## Do sprawdzenia
1. Przeczytaj `project.godot` - konfiguracja projektu
2. Znajdź wszystkie sceny (`**/*.tscn`)
3. Znajdź wszystkie skrypty (`**/*.gd`)
4. Sprawdź strukturę folderów

## Raport powinien zawierać
- Nazwa projektu i wersja Godot
- Lista scen i ich przeznaczenie
- Lista skryptów
- Brakujące elementy względem dokumentacji (`docs/00_godot_project_overview_for_ai.md`)

## Porównaj z Definition of Done
- [ ] Gra startuje i ładuje poziom
- [ ] Gracz się pojawia na starcie
- [ ] Da się poruszać
- [ ] Dotknięcie przeszkody = śmierć i restart
